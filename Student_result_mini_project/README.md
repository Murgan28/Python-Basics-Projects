# Student Result Management System (Python)

This is a beginner-friendly Python mini project that generates a student report card based on marks entered by the user.

# Project Description
The program takes student details such as name, age, roll number, date of birth, division, and marks of three subjects.  
It then calculates:
- Total marks
- Average marks
- Grade
- Pass/Fail result

Finally, it displays a formatted report card.
